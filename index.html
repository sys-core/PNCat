Bienvenue sur la page de PNCat

Nous vous présentons un outil réseau similaire à NetCat.

Comment l'utiliser

Sur un terminal ou un prompt shel cmd.exe, exécutez le script comme suit:

syscore$ ./PNCat.py -l -p 9999 -c

Vous avez préparé ainsi votre serveur PNCat.

Sur un autre terminal ou cmd.exe, exécutez votre script en mode client. Rappelez vous que votre script lit depuis stdin et le fait avant que le marqueur EOF est reçu. Pour envoyer EOF, tapez un CTRL-D sur votre clavier :

syscore$ ./PNCat.py -t localhost -p 9999 PNC:# ls -la
